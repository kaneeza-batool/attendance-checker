# Attendance Checker

A simple web app to calculate attendance percentage and show eligibility.

## Features

✔ Accepts total classes and attended classes  
✔ Calculates percentage  
✔ Shows result in Bootstrap modal

## How to Use

1. Open `index.html`
2. Enter total number of classes
3. Enter attended classes
4. Click **Check Attendance**

## Technologies Used

✔ HTML5  
✔ Bootstrap 5  
✔ JavaScript  
✔ CSS
